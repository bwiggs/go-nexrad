Go Tools for processing NEXRAD binary data.

## Features

- NEXRAD Level 2 (Archive II Format) Processing
	- Products 
		- Reflectivity (REF)
		- Velocity (VEL)
		- Spectrum Width (SW)
		- Correlation Coefficient (RHO)
		- Differential Reflectivity (ZDR)
